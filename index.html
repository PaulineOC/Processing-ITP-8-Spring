Processing-itp-8-spring

Download .zip Download .tar.gz View on GitHub

This game was supposed to be a simple “Welcome” to the spring and a farewell to the winter with the actual video game. I used the color tracking technique, so the player is a bird. In the background, the snow and rain fall. The bird must simply catch the sun and then slowly the bad weather will clear.

Link to original post: https://itp.nyu.edu/classes/icm-dano-spring2014/spring/
